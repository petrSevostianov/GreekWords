# Сочинения
Пишем сочинения на греческом языке, используя слова из нашего словаря. 
Сначала мы напишем набор тематических заготовок.
Используй эти заготовки для написания сочинений на разные темы.
Не используй предложения из заголовков если они не подходят по смыслу.
Заменяй такие предложения на более подходящие.
В кодблоках я буду указывать контекст/задание для сочинения.

Следует мнинмизировать использование сослагательного наклонения, прошедшего времени и сложных конструкций.

Αγαπητοί αναγνώστες,
Είμαι απόφοιτος και θέλω να σας πω πώς προετοιμάστηκα για τις εξετάσεις.

ωράριο:
