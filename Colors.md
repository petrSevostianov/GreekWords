# πράσινος
### <color=#00FF00>зеленый</color>

**Мужской род:** ед.ч. **πράσινος** / мн.ч. **πράσινοι**

**Женский род:** ед.ч. **πράσινη** / мн.ч. **πράσινες**

**Средний род:** ед.ч. **πράσινο** / мн.ч. **πράσινα**

### Примеры

- Ένα πράσινο μήλο.
*Зеленое яблоко.*
- Μια πράσινη μπλούζα.
*Зеленая кофта.*
- Οι πράσινοι κήποι είναι όμορφοι.
*Зеленые сады красивые.*
- Τα φύλλα είναι πράσινα την άνοιξη.
*Листья зеленые весной.*

### Родственные слова

- **πράσινο** — зеленый цвет
- **πρασινίζω** — зеленеть
- **πρασινάδα** — зелень
- **πρασινωπός / πρασινωπή / πρασινωπό** — зеленоватый
