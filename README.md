# ο άνθρωπος
человек, люди (в широком смысле)

### Формы
ед.ч.: **ο άνθρωπος** / **τον άνθρωπο** / **του ανθρώπου**

мн.ч.: **οι άνθρωποι** / **τους ανθρώπους** / **των ανθρώπων**

### Примеры
- Ο άνθρωπος είναι ψυχικό όν.
*Человек — существо духовное.*
- Βλέπω τον άνθρωπο στο δρόμο.
*Я вижу человека на дороге.*
- Η καρδιά του ανθρώπου είναι σημαντική.
*Сердце человека важно.*
- Οι άνθρωποι αγαπούν τη ζωή.
*Люди любят жизнь.*

### Запоминаем
Корень **anthropo-** встречается в английском и русском: *anthropology* / *антропология*, *philanthropist* / *филантроп*.

# η καρδιά
**Перевод:** сердце
**Часть речи:** существительное
**Род:** женский
> Корень **cardio-** знаком по английскому и русскому: *cardiology* / *кардиология*, *cardiac*.

### Формы

ед.ч.: **η καρδιά** / **την καρδιά** / **της καρδιάς**

мн.ч.: **οι καρδιές** / **τις καρδιές** / **των καρδιών**

### Примеры

- Η καρδιά μου χτυπά γρήγορα.
*Мое сердце бьется быстро.*
- Αγαπώ την καρδιά σου.
*Я люблю твое сердце.*
- Τα αισθήματα της καρδιάς είναι βαθιά.
*Чувства сердца глубоки.*
- Οι καρδιές των ανθρώπων είναι διαφορετικές.
*Сердца людей различны.*

# γράφω
**Перевод:** писать
**Часть речи:** глагол
**Группа:** 1 (оканчивается на -ω)
> Корень **graph-** знаком по английскому и русскому: *graphic* / *графика*, *geography* / *география*, *biography* / *биография*.

### Спряжение

Сингулярное: **γράφω** / **γράφεις** / **γράφει**

Множественное: **γράφουμε** / **γράφετε** / **γράφουν**

### Примеры

- Εγώ γράφω ένα γράμμα.
*Я пишу письмо.*
- Εσύ γράφεις τα ονόματα;
*Ты пишешь имена?*
- Αυτός γράφει ένα βιβλίο.
*Он пишет книгу.*
- Εμείς γράφουμε στο σημειωματάριο.
*Мы пишем в тетради.*
- Εσείς γράφετε με μολύβι ή στυλό;
*Вы пишете карандашом или ручкой?*
- Αυτοί γράφουν πολλά.
*Они много пишут.*


# αγαπώ
**Перевод:** любить
**Часть речи:** глагол
**Группа:** 2 (оканчивается на -ώ)
> Слово **agape** (ἀγάπη) вошло в богословский словарь многих европейских языков как обозначение безусловной любви.

### Спряжение

Сингулярное: **αγαπώ** / **αγαπάς** / **αγαπά**

Множественное: **αγαπάμε** / **αγαπάτε** / **αγαπούν**

### Примеры

- Εγώ αγαπώ τη ζωή.
*Я люблю жизнь.*
- Εσύ αγαπάς τον άνθρωπό σου;
*Ты любишь своего человека?*
- Αυτή αγαπά τα παιδιά της.
*Она любит своих детей.*
- Εμείς αγαπάμε τη φύση.
*Мы любим природу.*
- Εσείς αγαπάτε την ελληνική κουλτούρα.
*Вы любите греческую культуру.*
- Αυτοί αγαπούν τη μουσική.
*Они любят музыку.*


# μεγάλος
**Перевод:** большой, великий
**Часть речи:** прилагательное
> Корень **mega-** широко используется в английском и русском: *megabyte* / *мегабайт*, *megalopolis* / *мегаполис*, *megaphone* / *мегафон*.

### Формы по роду и числу

**Мужской род:** ед.ч. **μεγάλος** / мн.ч. **μεγάλοι**

**Женский род:** ед.ч. **μεγάλη** / мн.ч. **μεγάλες**

**Средний род:** ед.ч. **μεγάλο** / мн.ч. **μεγάλα**

### Примеры

- Ένας μεγάλος άνθρωπος.
*Большой человек.*
- Μια μεγάλη καρδιά.
*Большое сердце.*
- Ένα μεγάλο βιβλίο.
*Большая книга.*
- Μεγάλοι άνθρωποι.
*Большие люди.*
- Μεγάλες θάλασσες.
*Большие моря.*
- Μεγάλα διαλύματα.
*Большие разрешения.*
- Αυτός ο άνθρωπος είναι μεγάλος και σοφός.
*Этот человек большой и мудрый.*


# το νερό
**вода**
существительное, cредний род

### Формы

ед.ч.: **το νερό** / **το νερό** / **του νερού**

мн.ч.: **τα νερά** / **τα νερά** / **των νερών**

### Примеры

- Το νερό είναι απαραίτητο για τη ζωή.
*Вода необходима для жизни.*
- Πίνω νερό κάθε μέρα.
*Я пью воду каждый день.*

#### Запоминаем

Корень **nero-** встречается в названиях минеральных вод (напр. *Acqua Nero*). В русском слово не имеет прямого родства, но схоже с именем *Нерон* (хотя этимологически не связано).
