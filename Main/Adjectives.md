# σοβαρός
## серьёзный
- Ед: σοβαρός, σοβαρή, σοβαρό
- Мн: σοβαροί, σοβαρές, σοβαρά

### Наречие
- σοβαρά - серьёзно

### Примеры
*Это серьёзная проблема.* = *Αυτό είναι ένα σοβαρό πρόβλημα.*
*Он серьёзный человек.* = *Είναι σοβαρός άνθρωπος.*

# ομαδικός
## групповой
- Ед: ομαδικός, ομαδική, ομαδικό
- Мн: ομαδικοί, ομαδικές, ομαδικά

### Наречие
- ομαδικά - группами, коллективно

### Примеры
*Это групповое занятие.* = *Αυτό είναι ένα ομαδικό μάθημα.*
*Они работают коллективно.* = *Δουλεύουν ομαδικά.*

# αγνός

# χωρισμένος
## разведённый, раздельный
- Ед: χωρισμένος, χωρισμένη, χωρισμένο
- Мн: χωρισμένοι, χωρισμένες, χωρισμένα

*разведённый - отдельно живущий* = *χωρισμένος*

*разделяю, отделяю* = *χωρίζω*
*разлука, расставание* = *ο χωρισμός*
*отдельно* = *χωριστά*


*Он разведённый и живёт отдельно.* = *Είναι χωρισμένος και μένει χωριστά.*
*Данные разделены на две группы.* = *Τα δεδομένα είναι χωρισμένα σε δύο ομάδες.*
