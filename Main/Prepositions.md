# χωρίς
## без

### Примеры
*Кофе без сахара, пожалуйста.* = *Καφέ χωρίς ζάχαρη, παρακαλώ.*
*Он ушёл без куртки.* = *Έφυγε χωρίς μπουφάν.*
*Мы не можем жить без воды.* = *Δεν μπορούμε να ζήσουμε χωρίς νερό.*

# με
## с
*Еду поездом* = *Πηγαίνω με το τρένο.*
*Он говорит с другом.* = *Μιλάει με τον φίλο του.*

# σε
## в, на

# από
## от, из

# για
## для, о

# προς
## к, по направлению к

# κατά
## против, в соответствии с

# παρά
## рядом с, вопреки

# ανά
## per
*за килограмм* = *ανά κιλό*
*на человека* = *ανά άτομο*
*в неделю* = *ανά εβδομάδα*
*в день* = *ανά ημέρα / ανά μέρα*

*Я плачу десять евро за килограмм* = *Πληρώνω δέκα ευρώ ανά κιλό*
*Я хожу туда два раза в неделю* = *Πηγαίνω εκεί δύο φορές ανά εβδομάδα*

# μέχρι
## до; пока не

### Примеры
*Подожди до завтра.* = *Περίμενε μέχρι αύριο.*
*Мы работаем до пяти.* = *Δουλεύουμε μέχρι τις πέντε.*
*Не уходи, пока я не вернусь.* = *Μην φύγεις μέχρι να γυρίσω.*

# ως
## как; до; в качестве; as

### Примеры
*Он работает как врач.* = *Δουλεύει ως γιατρός.*
*Мы дошли до площади.* = *Φτάσαμε ως την πλατεία.*
*Я говорю это как друг.* = *Το λέω αυτό ως φίλος.*
