переводи на греческий

как = πως
как-то = κάπως
 
никогда = ποτέ

кто = ποιος/ποια/ποιο / ποιοι/ποιες/ποια
кому = σε ποιον/σε ποια/σε ποιο / σε ποιους/σε ποιες/σε ποια
от кого = από ποιον/από ποια/από ποιο / από ποιους/από ποιες/από ποια

для чего = για τι
для кого = για ποιον/για ποια/για ποιο / για ποιους/για ποιες/για ποια

что = τι
где = που
когда = πότε
почему = γιατί
Сколько = πόσος/πόση/πόσο / πόσοι/πόσες/πόσα
зачем = γιατί
чей = τίνος
где-то = κάπου
когда-то = κάποτε

какой = ποιος / ποια / ποιο
который = ο οποίος / η οποία / το οποίο или просто που
куда = πού
откуда = από πού
докуда / до куда = μέχρι πού
сколько раз = πόσες φορές
сколько лет = πόσων χρονών
сколько стоит = πόσο κάνει / πόσο κοστίζει
когда? во сколько? = πότε / τι ώρα
каким образом = με ποιον τρόπο
с кем = με ποιον / με ποια / με ποιους...
без кого / без чего = χωρίς
о ком / о чём = για ποιον / για τι
у кого = σε ποιον / ποιος έχει... по контексту
кого? = ποιον / ποια / ποιο
что за...? = τι ...; / τι είδους ...;
ли = часто просто интонация, иногда αν
разве? = μήπως / μα
